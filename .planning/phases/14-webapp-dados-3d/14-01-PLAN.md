- `webapp/js/dice.js` — Lógica Three.js y física de dados

## Task 4: Implementar Three.js (js/dice.js)

Create the JavaScript with Three.js implementation:

**Features:**
1. **Scene setup:**
   - Scene, camera (orbital), renderer
   - Iluminación ambiental + direccional para realismo
   - Floor/plano para que los dados rueden

2. **Dice geometry:**
   - Dos cubos con geometría Box (geometría de dado)
   - Tamaño proporcional para visualización

3. **Dice textures:**
   - Texturas con puntos de dado (1-6) usando canvas dinámico
   - Generar caras de dado programáticamente

4. **Physics/animation:**
   - Física simple: velocidad angular aleatoria + gravedad
   - Animación de lanzamiento con rotación realista
   - Dados se detienen mostrando valores aleatorios 1-6

5. **Interactivity:**
   - Botón "Lanzar" inicia animación
   - Detectar cuando dados se detienen
   - Calcular valores finales basados en rotación

6. **Telegram WebApp integration:**
   - Inicializar WebApp SDK
   - Obtener user_id desde initDataUnsafe
   - Enviar resultado al bot via sendData

**Files to create:**
- `webapp/js/dice.js`
