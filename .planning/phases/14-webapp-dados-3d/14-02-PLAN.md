Update `show_minigames_menu` and related handlers to launch the WebApp instead of using callback-based dice game.

**Changes in `handlers/gamification_user_handlers.py`:**

1. **Import WebAppInfo:**
```python
from aiogram.types import WebAppInfo
```

2. **Update `show_minigames_menu`:**
```python
keyboard = InlineKeyboardMarkup(inline_keyboard=[
    [InlineKeyboardButton(
        text="🎲 Lanzar dados",
        web_app=WebAppInfo(url=WEBAPP_URL)

    [InlineKeyboardButton(text="🔙 Volver", callback_data="back_to_main")]
])
```

3. **Add WEBAPP_URL configuration:**
- Development: `http://localhost:8080/webapp/dice.html`
