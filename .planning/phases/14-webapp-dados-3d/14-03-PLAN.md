Create validation logic in the WebApp data handler:

**Validations required:**
- Ambos valores están entre 1-6
- La suma está entre 2-12
- El resultado es consistente (no manipulado)

**Implementation:**
```python
def validate_dice_result(dice1: int, dice2: int) -> bool:
    """Validar que el resultado de los dados sea posible."""
    if not (1 <= dice1 <= 6 and 1 <= dice2 <= 6):
        return False
    if not (2 <= dice1 + dice2 <= 12):
        return False
    return True
```

## Task 2: Implementar lógica de victoria

Implement the win condition logic:

```python
def check_win(dice1: int, dice2: int) -> bool:
    # Ambos pares (2, 4, 6)
    both_even = dice1 % 2 == 0 and dice2 % 2 == 0
    # Dobles (mismo número)
    is_double = dice1 == dice2
    return both_even or is_double
```
