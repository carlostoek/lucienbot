# Utils

Utilidades y helpers.

## Archivos
- [helpers.py](helpers.py) - Funciones auxiliares
- [lucien_voice.py](lucien_voice.py) - Plantillas de voz de Lucien

## helpers.py
Funciones utilities generales:
```python
- format_besitos(amount)     # Formatear besitos
- parse_callback_data(data) # Parsear callback data
- validate_telegram_id(tg_id)  # Validar ID
```

## lucien_voice.py
Plantillas de mensaje con la voz de Lucien:
```python
- WELCOME = "Bienvenido al reino de Diana..."
- VIP_MESSAGE = "Lucien informa que este contenido..."
- DAILY_GIFT = "Diana ha dejado un regalo..."
```

## Reglas
- Funciones genéricas van aquí
- NO lógica de negocio
- Máximo 50 líneas por función

## Voz de Lucien
- 3ra persona ("Lucien gestiona...")
- Elegante, misterioso
- "Diana" como figura central
- "Visitantes" no "usuarios"
- "Custodios" no "admins"
